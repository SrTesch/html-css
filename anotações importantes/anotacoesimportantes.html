Um pouco sobre a hierarquia de títulos

When you read o conteúdo de um site, você vai precisar colocar os títulos de formar organizada!
Pra fazer isso em html, você pode usar as tags <h1>, <h2>, <h3>, e assim vai até o <h6>.
E só pra ficar claro.. H1 vai ser o primeiro título e H2 um subtítulo. H3 será um subtítulo de H2, assim por diante

Comandos de formatação

  • The tag <pre> i for show exactly text how you read in the space bellow that

Listas

Ordered lists

The ordered lists are read using the tag <ol> and you HAVE TO USE </ol> to close then.
To improve the itens, you have to use the tag <li> and the close tag is opitional.

The lists can be ordinated by numbers, letters and roman numbers... For that you have to use " type='' " inside the tag <ol>.
When you use the 'type', you choice the ordinating type for your list. Using 'A' for big letters, 'a' for small letters, 'I' for big roman numbers and 'i' for small roman numbers

Unordered lists

The Unordered lists are read using the tag <ul> and you HAVE TO USE </ul> to close then.
Like in the ordered lists, on Unordered lists, you can use <li> to improve the itens.
The default is an disc, but you can change this form using "type='' " inside the tag.
'Disc' is the default form, 'circle' if you want to use a circle and 'square' if you want a square

Links

You can use the tag <a> to improve a hyperlink in the especifc text that you mark.
If the hyperlink is a other website, is recommended to use the "target='_blank' rel='external' " to show the link in an new tab ;)

Audio

Usando a tag <audio> você pode implementar audios no seu site.
Dentro da tag voce pode usar o comando "autoplay", porém este comando nao funciona na marioria dos navegadores.
portanto o ideal é que você use a tag "controls" para o cliente poder clicar no botão do play
e interagir com o site e ouvir a música
